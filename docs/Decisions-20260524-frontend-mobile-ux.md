| 항목 | 내용 |

| 긴급도 | 높음 |
| 상태 | 결정완료 |

**배경**

**결정**

1. `html { overflow-x: hidden }` — 문서 최상위
2. `.content { overflow-x: hidden }` (모바일 미디어쿼리)
3. `.strategy-layout { overflow-x: hidden }` + `.strategy-grid { min-width: 0 }`

**이유**
근본 원인: flex item의 `min-width: auto` 기본값이 내용물(1311px) 크기로 팽창 → 부모 전체 확장. `min-width: 0`으로 차단. `overflow-x: clip`은 Safari 16 미만 미지원으로 `hidden`으로 교체
